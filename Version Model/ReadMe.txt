โมเดลทั้งหมดนี้ จัดทำขึ้นเพื่อให้เหมาะสมต่อการใช้งานในพื้นที่ทำงานที่เป็นพื้นที่เรียบ หรือหรือขรุขระเล็กน้อยเช่น หน้าดินปลูกพืชผักที่ไม่มีหญ้าทึบหนาโดย แต่ละโมเดลมีขอบเขตและข้อจำกัด ดังนี้
Version 2
-	โครงสร้างของหุ่นยนต์ ไม่ซับซ้อน เข้าใจการทำงานง่าย รับน้ำหนักได้มาก
-	มีข้อจำกัดคือ เหมาะสำหรับแขนหุ่นยนต์ที่ สามารถเคลื่อนที่ในแนวขึ้นลงได้ หากเคลื่อนที่ในแนวราบอาจไม่เหมาะ
Version 3
-	โครงสร้างของหุ่นยนต์ มีขนาดเล็กลงจากเดิม แต่ยังสามารถ รับน้ำหนักได้มาก
-	มีข้อจำกัดคือ เหมาะสำหรับแขนหุ่นยนต์ที่ สามารถเคลื่อนที่ในแนวขึ้นลงได้ หากเคลื่อนที่ในแนวราบอาจไม่เหมาะ
Version 4
-	โครงสร้างของหุ่นยนต์มีความซับซ้อนมากขึ้น(ส่งผลต่อการซื้ออุปกรณ์มากขึ้น)ทำงานในที่แคบได้  ยังสามารถ รับน้ำหนักได้มาก
-	มีข้อจำกัดคือ เหมาะสำหรับแขนหุ่นยนต์ที่ สามารถเคลื่อนที่ในแนวขึ้นลงได้ หากเคลื่อนที่ในแนวราบอาจไม่เหมาะ
Version 5
-	โครงสร้างของหุ่นยนต์ ไม่ซับซ้อน เข้าใจการทำงานง่าย รับน้ำหนักได้น้อย เหมาะกับแขนหุ่นยนต์ที่หลากหลาย
-	มีข้อจำกัดคือ เหมาะงานที่ไม่เน้นปริมาณ เพราะอาจจะต้องมีการเปลี่ยนภาชนะใส่หลายหลัง หรือหาต้องการให้ระบบวนกลับไปยังจุดเปลี่ยนภาชนะ ก็อาจจะยากต่อการเขียนโปรแกรม
Version 6
-	โครงสร้างของหุ่นยนต์รถค่อนข้างใหญ่ เก็บชิ้นงานได้มากที่สุด แต่โครงสร้างซับซ้อน วัสดุที่ใช้จึงมากไปด้วย
-	มีข้อจำกัดคือ เหมาะพื้นที่การทำงานที่ใหญ่ และไม่สามารถเก็บชิ้นงานที่ต่ำกว่ารถได้
Version 7
-	โครงสร้างของหุ่นยนต์ ไม่ซับซ้อน น้ำหนักรถเบา ขนาดไม่ใหญ่จนเกินไป รับน้ำหนักได้พอดี
-	มีข้อจำกัดคือ มีพื้นที่สำหรับวางกล่องอุปกรณ์ที่น้อย จึงทำให้ดูเกะกะสายตา
Version 8
-	โครงสร้างของหุ่นยนต์ซับซ้อนเล็กน้อย เข้าใจการทำงานง่าย รับน้ำหนักได้มาก เหมาะกับแขนหุ่นยนต์ที่หลากหลาย
-	มีข้อจำกัดคือ เหมาะพื้นที่การทำงานที่ใหญ่ และไม่สามารถเก็บชิ้นงานที่ต่ำกว่ารถได้
Version 9
-	โครงสร้างของหุ่นยนต์ซับซ้อนเล็กน้อย แต่ขนาดไม่ใหญ่เทอะทะ แข็งแรง แต่มีพื้นที่สำหรับวางกล่องอุปกรณ์ที่น้อย
-	มีข้อจำกัดคือ โครงสร้างยากต่อการทำ และเสี่ยงต่อการทำให้ไม่บาลานซ์
Version 10
-	โครงสร้างของหุ่นยนต์ ไม่ซับซ้อน เข้าใจการทำงานง่าย รับน้ำหนักได้มาก และบาลานซ์ ผ่านความคิดเห็นจากทีมงานว่าเห็นสมควร
-	มีข้อจำกัดคือ โครงสร้าซับซ้อนเล็กน้อย แต่สามารถควบคุมปริมาณได้ ยังไม่มารถใช้ในพื้นที่การทำงานเล็กได้

ข้อเสนอะแนะ เลือกใช้โครงสร้าง จัดสรรอุปกรณ์ให้เหมาะสม คำนวณงบประมาณ เน้นใช้งานได้จริงเหมาะสมกับงานคุณภาพของหุ่นยนต์จึงจะแสดงออกอย่างเต็มที่
